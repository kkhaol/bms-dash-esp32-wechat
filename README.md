# X仪表小程序

这是 X仪表 的微信小程序和配套仪表固件项目。小程序用于连接 X仪表、选择要使用的蚂蚁 BMS，并查看核心电池状态。

手机只负责配置和查看状态，不会直接连接 BMS，也不需要长期保持在前台。

## 小程序能做什么

- 连接 X仪表。
- 查看 X仪表 和 BMS 是否已连接。
- 查看 SOC、电压、电池温度。
- 扫描附近 BMS。
- 从附近 BMS 或历史 BMS 中选择目标。
- 保存选择结果，让 X仪表 下次开机优先连接上次使用的 BMS。
- 调整高级设置，例如屏幕亮度、低电量提醒阈值、调试日志开关。

## 页面说明

### 仪表

首页只显示最重要的信息：

- X仪表 是否连接。
- BMS 是否连接。
- 当前选择的 BMS。
- SOC、电压、电池温度。

未连接时点击“连接X仪表”。已连接 X仪表 但未连接 BMS 时，点击“选择BMS”进入 BMS 页面。

### BMS

这里用于扫描和选择电池。

1. 点击“扫描附近BMS”。
2. 在卡片中选择目标 BMS。
3. 点击“连接此BMS”。

连接成功后，这个 BMS 会成为下次开机优先连接的目标。

### 设置

设置页放高级功能：

- 自动连接上次BMS。
- 优先连接信号最强BMS。
- 屏幕亮度。
- 低电量提醒阈值。
- 调试日志。
- 恢复默认设置。

调试日志默认关闭，正常使用不需要打开。

## 使用方法

1. 用微信开发者工具导入 `miniprogram/` 文件夹。
2. 本地测试可以使用测试号或自己的小程序 AppID。
3. 请用真机调试，模拟器通常不能完整测试蓝牙。
4. 打开手机蓝牙，并允许小程序使用蓝牙权限。
5. 在“仪表”页点击“连接X仪表”。
6. 到“BMS”页扫描并选择要连接的 BMS。
7. 回到“仪表”页查看 SOC、电压和电池温度。

## 固件目录

- `esp32/ANT_BMS_TFT_Dashboard_V2_2_StateMachine/`：正式状态机版本。
- `esp32/ANT_BMS_TFT_Dashboard_V2_2test/`：临时测试版本。

正常使用建议烧录正式状态机版本。
