# X仪表

X仪表是一款用于查看电池状态的小仪表。

## 主要功能

- 开机后自动连接上次使用的电池。
- 如果上次电池未连接成功，会自动搜索附近电池并持续重试。
- 主界面显示电量、温度、功率、压差、电压、电流和剩余容量。
- 顶部图标显示电池连接状态和手机连接状态。
- 连接中会显示简洁的动态圆环，避免误以为设备卡住。
- 未获取到电池数据时会自动重新连接。

## 当前界面

- 左上角显示温度。
- 右上角显示蓝牙和手机状态图标。
- 左侧显示功率和压差。
- 中间显示电量圆环。
- 右侧显示电压和电流。
- 底部显示剩余容量。

## 使用提示

- 通电后请等待仪表自动连接电池。
- 如果长时间没有数据，请确认电池已开启并靠近仪表。
- 如果电池刚被其它设备连接，仪表可能需要等待一段时间后重新连接。
