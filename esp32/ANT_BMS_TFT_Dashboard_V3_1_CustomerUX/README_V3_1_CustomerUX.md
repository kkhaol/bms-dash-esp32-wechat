# ANT_BMS_TFT_Dashboard_V3_1_CustomerUX

V3.1 继续面向客户体验优化 170x320 ST7789 横屏仪表，重点修复 V3.0 的自动连接状态闪切和主界面信息比例问题。

## 本版重点

- 开机仍先尝试上次保存的 BMS，失败后直接进入附近最强 BMS 自动搜索和连接。
- 自动连接失败后持续执行“扫描 -> 选择最强 -> 连接 -> 再扫描”的循环，不再频繁跳到等待手机操作页。
- 连续多轮完全找不到 BMS 时才显示“等待手机操作”，后台仍会周期性重试。
- 保存的 MAC 找不到时，会用上次保存的真实蓝牙名称辅助匹配，适配蓝牙随机地址变化。
- 主界面保持顶部温度和连接图标、左侧电压/电流、中间 SOC 圆环。
- 电流显示改为固定字号，避免 `0.0` 显得过大。
- SOC 数字缩小，保留圆环呼吸感。
- 右侧改为 4 行图标数据：总容量、剩余容量、最高单体电压、最低单体电压。
- 所有新增数据均来自真实 BMS 解析字段，没有写死示例值。

## 编译

arduino-cli compile --fqbn esp32:esp32:esp32:PartitionScheme=huge_app Projects\ANT_BMS_TFT_Dashboard_V3_1_CustomerUX
