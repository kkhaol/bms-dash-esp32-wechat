# ANT-BMS 小仪表 V2.5 客户体验版

这是面向客户体验的正式 UI 版本。屏幕继续使用 ST7789、170x320、横屏 320x170，TFT_eSPI 配置沿用 V2.4.3 已验证的 `build_opt.h` 和 `tft_setup.h`。

## 本版重点

- 主界面按参考图重做：左侧电压/电流，中间 SOC 圆环，右侧总容量/剩余容量，顶部温度、蓝牙和手机状态图标。
- 移除主界面的“状态正常”、MOS、压差、工程状态名和大功率数字。
- 开机和连接状态统一为小猫主题动画。
- 中文点阵字体覆盖本版 UI 所有中文，避免乱码或缺字。
- 连接失败后持续自动重试：先尝试上次保存电池，再尝试附近信号最强电池。
- 保留现有 BMS 数据解析、绑定存储、微信小程序通信协议和 TFT_eSPI ST7789 引脚配置。

## 验证

- 已使用 ESP32 core 2.0.11、TFT_eSPI 2.5.43、NimBLE-Arduino 2.5.0 编译通过。
- 已刷入 COM6，并通过串口确认 `BOOT_OK`、TFT 初始化、Sprite 创建和自动重试日志。
