# ANT-BMS 小仪表 V3.0 客户体验版

这是按 320x170 横屏重新整理显示层的客户体验版。屏幕配置继续沿用已验证的 ST7789、TFT_eSPI、GPIO18/23/5/16/17、rotation=1、invert=true。

## 本版重点

- 主界面按固定坐标重画：顶部状态、左侧电压/电流、中间 SOC 圆环、右侧容量卡片。
- 中文点阵字体重新生成，覆盖本版所有中文，并使用固定 13x15 单元避免裁切。
- 状态页取消小猫和二轮车动画，改为抽象 loading 圆环、雷达扫描和手机呼吸点。
- 扫描阶段按短时间片刷新 UI，连接失败后持续重试。
- 保留原有 BMS 数据读取、解析、选择、保存和小程序通信协议。

## 验收重点

- 开机必须先显示界面，串口输出 `BOOT_OK`。
- 主界面不显示 MOS、状态正常、RSSI、MAC、UUID、BLE 名称或工程状态。
- 小程序电池列表显示真实蓝牙名称，不再显示“电池1 / 电池2”。
