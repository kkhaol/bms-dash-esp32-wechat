# X仪表 V3.5 闪烁修正版

这是本地版本，不推送 GitHub。本版只做最小修复，不改变主界面风格、整体布局、连接逻辑和数据读取逻辑。

## 本次修改

- 顶部状态栏不再整条刷新 `320x40` 区域。
- 温度区域改为单独刷新：`x=0, y=0, w=105, h=32`。
- 蓝牙和手机图标区域改为单独刷新：`x=240, y=0, w=80, h=32`。
- 顶部中间区域不再被刷新，避免擦到 SOC 圆环上方。
- 主界面刷新间隔从约 33ms 调整为约 120ms，降低无意义刷屏压力。
- SOC 数字和 `%` 同步下移 3px，让数字更接近圆环视觉中心。

## 保持不变

- UI 风格和整体布局不变。
- 左右数据框、颜色、分割线不变。
- 连接页不变。
- 小程序逻辑不变。
- BMS 连接、解析、读取和自动重试逻辑不变。
